# Bare-metal STM32H7 — Course index (Nucleo-H723ZG)

This folder holds the written lessons for a bare-metal programming track using **STM32CubeIDE** and the **NUCLEO-H723ZG** (STM32H723). Open chapters in order.

| Chapter | Title | File |
|--------|--------|------|
| **00a** | Required documents (RM, DS, board manual) | [00a_required_documents.md](00a_required_documents.md) |
| **00b** | STM32CubeIDE workspace and minimal project | [00b_stm32cubeide_setup.md](00b_stm32cubeide_setup.md) |
| **00c** | Chip headers — CMSIS device headers in the project | [00c_chip_headers.md](00c_chip_headers.md) |

**Figures:** screenshots for each chapter live under `images/00a/`, `images/00b/`, and `images/00c/` at the **repository root** (next to the `docs/` folder), so paths in the markdown use `../images/...` from here.

---

## Suggested reading path

1. Collect PDFs (**00a**).  
2. Create the IDE project and confirm a clean build (**00b**).  
3. Add CMSIS **Include** + **Device** headers, wire include paths, and select the exact H7 part (**00c**).

Later chapters can be appended to this table as you add them.
